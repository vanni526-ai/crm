# 课程接口文档

## 文档概述

本文档为前端APP开发团队提供课程相关接口的完整说明，包括接口路径、请求参数、响应字段和调用示例。所有接口基于tRPC协议，支持类型安全的端到端通信。

**API基础URL**: `https://crm.bdsm.com.cn/api`

**更新日期**: 2026-02-27

---

## 核心接口列表

| 接口名称 | 接口路径 | 权限要求 | 功能说明 |
|---------|---------|---------|---------|
| 获取课程列表 | `courses.list` | 公开 | 获取所有可预约的课程列表 |
| 获取单个课程详情 | `courses.getById` | 公开 | 根据课程ID获取详细信息 |
| 创建课程 | `courses.create` | 需登录（管理员） | 创建新课程 |
| 更新课程 | `courses.update` | 需登录（管理员） | 更新课程信息 |
| 删除课程 | `courses.delete` | 需登录（管理员） | 删除课程 |

## 1. 获取课程列表

### 接口信息

- **接口路径**: `courses.list`
- **请求方法**: `query`
- **权限要求**: 公开接口，无需登录

### 请求参数

```typescript
{
  isBookable?: boolean;  // 可选：是否只返回可预约的课程（默认true）
  level?: "入门" | "深度" | "订制" | "剧本";  // 可选：按课程程度筛选
```

### 前端调用示例（React Native + tRPC）

```typescript
import { trpc } from '@/lib/trpc';

function CourseListScreen() {
  // 获取所有可预约的课程
  const { data, isLoading, error } = trpc.courses.list.useQuery({
    isBookable: true
  });

  if (isLoading) return <Text>加载中...</Text>;
  if (error) return <Text>加载失败: {error.message}</Text>;

  return (
    <FlatList
      data={data?.data}
      renderItem={({ item }) => (
        <View>
          {/* 优先显示别名，如果没有别名则显示课程名称 */}
          <Text style={styles.courseName}>
            {item.alias || item.name}
          </Text>
          <Text style={styles.introduction}>{item.introduction}</Text>
          <Text style={styles.price}>¥{item.price}</Text>
          <Text style={styles.duration}>{item.duration}小时</Text>
          <Text style={styles.level}>{item.level}</Text>
        </View>
      )}
      keyExtractor={(item) => item.id.toString()}
    />
  );
}
```

### 前端调用示例（React Native + tRPC）

```typescript
import { trpc } from '@/lib/trpc';

function CourseDetailScreen({ route }) {
  const { courseId } = route.params;
  
  const { data, isLoading, error } = trpc.courses.getById.useQuery({
    id: courseId
  });

  if (isLoading) return <Text>加载中...</Text>;
  if (error) return <Text>加载失败: {error.message}</Text>;

  const course = data?.data;

  return (
    <ScrollView>
      {/* 优先显示别名 */}
      <Text style={styles.title}>{course.alias || course.name}</Text>
      {/* 如果有别名，在副标题显示完整课程名 */}
      {course.alias && (
        <Text style={styles.subtitle}>完整名称：{course.name}</Text>
      )}
      <Text style={styles.introduction}>{course.introduction}</Text>
      <Text style={styles.description}>{course.description}</Text>
      <View style={styles.infoRow}>
        <Text>价格：¥{course.price}</Text>
        <Text>时长：{course.duration}小时</Text>
        <Text>程度：{course.level}</Text>
      </View>
      <Button title="立即预约" onPress={() => handleBooking(course.id)} />
    </ScrollView>
  );
}
```

## 字段说明

### alias（课程别名）字段详解

**用途**: 为课程提供一个简短、易记的别名，方便前端APP显示使用。

**使用场景**:
1. **列表页显示**: 在课程列表中，优先显示`alias`字段，如果`alias`为空则显示`name`字段
2. **详情页标题**: 在课程详情页，将`alias`作为主标题，`name`作为副标题（如果两者不同）
3. **搜索和筛选**: 支持按`alias`或`name`搜索课程

**示例对比**:
- **完整课程名**: `"BDSM理论课程：从入门到精通"`
- **课程别名**: `"理论基础"`
- **前端显示**: 优先显示"理论基础"，用户点击后在详情页看到完整名称

**数据类型**: `string | null`（可为空）

**最大长度**: 100字符

## 课程程度说明

| 程度 | 说明 | 适合人群 |
|------|------|---------|
| `入门` | 基础课程 | 零基础或初学者 |
| `深度` | 进阶课程 | 有一定基础的学员 |
| `订制` | 定制课程 | 根据学员需求定制 |
| `剧本` | 剧本课程 | 特定场景或角色扮演 |

## 错误处理

所有接口在发生错误时返回以下格式：

```json
{
  "success": false,
  "error": {
    "code": "NOT_FOUND",
    "message": "课程不存在"
  }
}
```

**常见错误码**:
- `NOT_FOUND`: 课程不存在
- `UNAUTHORIZED`: 未登录或权限不足
- `BAD_REQUEST`: 请求参数错误
- `INTERNAL_SERVER_ERROR`: 服务器内部错误

## 前端集成配置

### tRPC客户端配置

```typescript
// lib/trpc.ts
import { createTRPCReact } from '@trpc/react-query';
import { httpBatchLink } from '@trpc/client';
import type { AppRouter } from '../../../server/routers';

export const trpc = createTRPCReact<AppRouter>();

export const trpcClient = trpc.createClient({
  links: [
    httpBatchLink({
      url: 'https://crm.bdsm.com.cn/api/trpc',
      headers: () => {
        // 如果需要认证，添加Token
        const token = getAuthToken();
        return token ? { Authorization: `Bearer ${token}` } : {};
      },
    }),
  ],
});
```

### React Native应用入口配置

```typescript
// App.tsx
import { QueryClient, QueryClientProvider } from '@tanstack/react-query';
import { trpc, trpcClient } from './lib/trpc';

const queryClient = new QueryClient();

export default function App() {
  return (
    <trpc.Provider client={trpcClient} queryClient={queryClient}>
      <QueryClientProvider client={queryClient}>
        <YourApp />
      </QueryClientProvider>
    </trpc.Provider>
  );
}
```
