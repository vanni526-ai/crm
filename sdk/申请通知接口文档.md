# 申请通知接口文档

> 本文档面向前端App（React Native）开发团队，详细说明"申请通知"功能的全部接口规范、请求/响应格式和集成方法。

---

## 功能概述

"申请通知"功能允许App用户向后台管理系统提交留言、建议、投诉、咨询或申请。管理员可在后台查看、回复、标记已读、归档或删除通知。App用户也可以查询自己提交的留言列表及管理员的回复。

---

## 数据结构

### 通知类型（NotificationType）

| 值 | 说明 |
|---|---|
| `general` | 一般留言（默认） |
| `complaint` | 投诉 |
| `suggestion` | 建议 |
| `consultation` | 咨询 |
| `application` | 申请 |

### 通知状态（NotificationStatus）

| 值 | 说明 |
|---|---|
| `unread` | 未读（初始状态） |
| `read` | 已读 |
| `replied` | 已回复 |
| `archived` | 已归档 |

### 通知详情对象（UserNotification）

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | `number` | 通知ID |
| `userId` | `number` | 提交用户ID |
| `userName` | `string \| null` | 用户名称 |
| `userPhone` | `string \| null` | 用户手机号 |
| `type` | `NotificationType` | 留言类型 |
| `title` | `string \| null` | 标题 |
| `content` | `string` | 留言内容 |
| `status` | `NotificationStatus` | 当前状态 |
| `adminReply` | `string \| null` | 管理员回复内容 |
| `repliedBy` | `number \| null` | 回复人ID |
| `repliedAt` | `string \| null` | 回复时间（ISO格式） |
| `readAt` | `string \| null` | 已读时间（ISO格式） |
| `createdAt` | `string` | 创建时间（ISO格式） |
| `updatedAt` | `string` | 更新时间（ISO格式） |

## 接口列表

### 1. 提交留言（App用户）

**接口路径：** `notifications.submit`  
**请求方式：** mutation（POST）  
**认证要求：** 无需登录（publicProcedure），但需传入userId

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `userId` | `number` | 是 | 当前登录用户的ID |
| `userName` | `string` | 否 | 用户名称（冗余字段，方便后台展示） |
| `userPhone` | `string` | 否 | 用户手机号（冗余字段） |
| `type` | `NotificationType` | 否 | 留言类型，默认 `general` |
| `title` | `string` | 否 | 标题，最多200字 |
| `content` | `string` | 是 | 留言内容，1-5000字 |

**响应格式：**

```json
{
  "success": true,
  "id": 1
}
```

**SDK调用示例：**

```typescript
const result = await api.notifications.submit({
  userId: currentUser.id,
  userName: currentUser.name,
  userPhone: currentUser.phone,
  type: 'suggestion',
  title: '关于课程安排的建议',
  content: '希望能增加周末的课程时段...',
});

if (result.success) {
  Alert.alert('提交成功', `留言ID: ${result.id}`);
}
```

### 2. 查询我的留言列表（App用户）

**接口路径：** `notifications.myList`  
**请求方式：** query（GET）  
**认证要求：** 无需登录（publicProcedure），但需传入userId

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `userId` | `number` | 是 | 当前登录用户的ID |
| `page` | `number` | 否 | 页码，默认1 |
| `pageSize` | `number` | 否 | 每页条数，默认20，最大50 |

**响应格式：**

```json
{
  "items": [
    {
      "id": 1,
      "userId": 100,
      "userName": "张三",
      "userPhone": "13800138000",
      "type": "suggestion",
      "title": "关于课程安排的建议",
      "content": "希望能增加周末的课程时段...",
      "status": "replied",
      "adminReply": "感谢您的建议，我们会认真考虑。",
      "repliedBy": 1,
      "repliedAt": "2026-02-06T10:30:00.000Z",
      "readAt": "2026-02-06T10:00:00.000Z",
      "createdAt": "2026-02-06T09:00:00.000Z",
      "updatedAt": "2026-02-06T10:30:00.000Z"
    }
  ],
  "total": 1,
  "page": 1,
  "pageSize": 20
}
```

**SDK调用示例：**

```typescript
const result = await api.notifications.myList(currentUser.id, {
  page: 1,
  pageSize: 10,
});

console.log(`共 ${result.total} 条留言`);
result.items.forEach(item => {
  console.log(`[${item.type}] ${item.title || '无标题'} - ${item.status}`);
  if (item.adminReply) {
    console.log(`管理员回复: ${item.adminReply}`);
  }
});
```

const api = createApiClient({

### 2. 提交留言页面示例

```tsx
import React, { useState } from 'react';
import { View, Text, TextInput, TouchableOpacity, Alert, ScrollView } from 'react-native';

const TYPES = [
  { value: 'general', label: '一般留言' },
  { value: 'complaint', label: '投诉' },
  { value: 'suggestion', label: '建议' },
  { value: 'consultation', label: '咨询' },
  { value: 'application', label: '申请' },
];

export default function SubmitNotificationScreen({ currentUser }) {
  const [type, setType] = useState('general');
  const [title, setTitle] = useState('');
  const [content, setContent] = useState('');
  const [loading, setLoading] = useState(false);

  const handleSubmit = async () => {
    if (!content.trim()) {
      Alert.alert('提示', '请输入留言内容');
      return;
    }

    setLoading(true);
    try {
      const result = await api.notifications.submit({
        userId: currentUser.id,
        userName: currentUser.name,
        userPhone: currentUser.phone,
        type: type as any,
        title: title.trim() || undefined,
        content: content.trim(),
      });

      if (result.success) {
        Alert.alert('提交成功', '您的留言已提交，我们会尽快处理');
        setTitle('');
        setContent('');
      }
    } catch (error) {
      Alert.alert('提交失败', error.message);
    } finally {
      setLoading(false);
    }
  };

  return (
    <ScrollView style={{ flex: 1, padding: 16 }}>
      {/* 类型选择 */}
      <Text style={{ fontSize: 14, color: '#666', marginBottom: 8 }}>留言类型</Text>
      <View style={{ flexDirection: 'row', flexWrap: 'wrap', gap: 8, marginBottom: 16 }}>
        {TYPES.map(t => (
          <TouchableOpacity
            key={t.value}
            onPress={() => setType(t.value)}
            style={{
              paddingHorizontal: 12, paddingVertical: 6,
              borderRadius: 16,
              backgroundColor: type === t.value ? '#3B82F6' : '#F3F4F6',
            }}
          >
            <Text style={{ color: type === t.value ? '#FFF' : '#374151' }}>{t.label}</Text>
          </TouchableOpacity>
        ))}
      </View>

      {/* 标题 */}
      <Text style={{ fontSize: 14, color: '#666', marginBottom: 8 }}>标题（可选）</Text>
      <TextInput
        value={title}
        onChangeText={setTitle}
        placeholder="请输入标题"
        maxLength={200}
        style={{ borderWidth: 1, borderColor: '#E5E7EB', borderRadius: 8, padding: 12, marginBottom: 16 }}
      />

      {/* 内容 */}
      <Text style={{ fontSize: 14, color: '#666', marginBottom: 8 }}>留言内容</Text>
      <TextInput
        value={content}
        onChangeText={setContent}
        placeholder="请输入留言内容..."
        multiline
        numberOfLines={6}
        maxLength={5000}
        textAlignVertical="top"
        style={{ borderWidth: 1, borderColor: '#E5E7EB', borderRadius: 8, padding: 12, minHeight: 150, marginBottom: 16 }}
      />

      {/* 提交按钮 */}
      <TouchableOpacity
        onPress={handleSubmit}
        disabled={loading}
        style={{
          backgroundColor: loading ? '#9CA3AF' : '#3B82F6',
          borderRadius: 8, padding: 14, alignItems: 'center',
        }}
      >
        <Text style={{ color: '#FFF', fontSize: 16, fontWeight: '600' }}>
          {loading ? '提交中...' : '提交留言'}
        </Text>
      </TouchableOpacity>
    </ScrollView>
  );
}
```

### 3. 我的留言列表页面示例

```tsx
import React, { useEffect, useState } from 'react';
import { View, Text, FlatList, TouchableOpacity, RefreshControl } from 'react-native';

const STATUS_LABELS = {
  unread: '待处理',
  read: '已读',
  replied: '已回复',
  archived: '已归档',
};

const STATUS_COLORS = {
  unread: '#F59E0B',
  read: '#3B82F6',
  replied: '#10B981',
  archived: '#6B7280',
};

export default function MyNotificationsScreen({ currentUser }) {
  const [items, setItems] = useState([]);
  const [total, setTotal] = useState(0);
  const [page, setPage] = useState(1);
  const [refreshing, setRefreshing] = useState(false);

  const loadData = async (p = 1) => {
    try {
      const result = await api.notifications.myList(currentUser.id, { page: p, pageSize: 20 });
      if (p === 1) {
        setItems(result.items);
      } else {
        setItems(prev => [...prev, ...result.items]);
      }
      setTotal(result.total);
      setPage(p);
    } catch (error) {
      console.error('加载留言列表失败:', error);
    }
  };

  useEffect(() => { loadData(); }, []);

  const onRefresh = async () => {
    setRefreshing(true);
    await loadData(1);
    setRefreshing(false);
  };

  const renderItem = ({ item }) => (
    <TouchableOpacity
      style={{ padding: 16, borderBottomWidth: 1, borderBottomColor: '#F3F4F6' }}
    >
      <View style={{ flexDirection: 'row', justifyContent: 'space-between', marginBottom: 4 }}>
        <Text style={{ fontWeight: '600' }}>{item.title || '无标题'}</Text>
        <Text style={{ color: STATUS_COLORS[item.status], fontSize: 12 }}>
          {STATUS_LABELS[item.status]}
        </Text>
      </View>
      <Text numberOfLines={2} style={{ color: '#6B7280', fontSize: 14 }}>{item.content}</Text>
      {item.adminReply && (
        <View style={{ marginTop: 8, padding: 8, backgroundColor: '#F0FDF4', borderRadius: 6 }}>
          <Text style={{ fontSize: 12, color: '#166534' }}>管理员回复: {item.adminReply}</Text>
        </View>
      )}
      <Text style={{ color: '#9CA3AF', fontSize: 12, marginTop: 4 }}>
        {new Date(item.createdAt).toLocaleString('zh-CN')}
      </Text>
    </TouchableOpacity>
  );

  return (
    <FlatList
      data={items}
      renderItem={renderItem}
      keyExtractor={item => item.id.toString()}
      refreshControl={<RefreshControl refreshing={refreshing} onRefresh={onRefresh} />}
      onEndReached={() => { if (items.length < total) loadData(page + 1); }}
      onEndReachedThreshold={0.3}
      ListEmptyComponent={
        <View style={{ alignItems: 'center', paddingVertical: 60 }}>
          <Text style={{ color: '#9CA3AF', fontSize: 16 }}>暂无留言</Text>
        </View>
      }
    />
  );
}
```

## 注意事项

1. `submit` 和 `myList` 接口为公开接口（publicProcedure），App用户无需管理员权限即可调用，但需要传入正确的 `userId`。

2. `list`、`detail`、`markRead`、`batchMarkRead`、`reply`、`archive`、`delete`、`unreadCount` 接口为受保护接口（protectedProcedure），需要通过 Token 认证。

3. 留言内容最大长度为5000字，标题最大长度为200字。

4. 建议在App中定期轮询 `unreadCount` 接口（如每30秒一次），用于在导航栏显示未读角标。

5. 用户提交留言后，建议同时传入 `userName` 和 `userPhone` 冗余字段，方便后台管理员快速识别用户。
