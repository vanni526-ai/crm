# 城市地图销售额占比可视化实现结果

## 实现内容

### 1. 销售额占比计算
- 计算每个城市销售额占总销售额的百分比
- 公式: `salesPercentage = (city.totalSales / totalSales) * 100`

### 2. 颜色深浅映射
- 使用蓝色渐变表示销售额占比
- 浅蓝色 `rgb(59, 130, 246)` → 深蓝色 `rgb(16, 185, 248)`
- 占比越高,颜色越深
- 计算公式:
  ```javascript
  const intensity = Math.min(salesPercentage / maxSalesPercentage, 1);
  const r = Math.floor(59 + (16 - 59) * intensity);
  const g = Math.floor(130 + (185 - 130) * intensity);
  const b = Math.floor(246 + (248 - 246) * intensity);
  ```

### 3. Tooltip增强
- 鼠标悬停时显示:
  - 城市名称
  - 订单数
  - 销售额
  - **销售占比(新增)**
  - 利润

### 4. 图例更新
- 底部图例说明:"颜色深浅代表销售额占比"
- 显示渐变色块示意(浅蓝→深蓝)
- 保留"气泡大小代表订单数量"说明

## 功能特点

1. **直观的视觉对比** - 通过颜色深浅快速识别销售额贡献度
2. **双重信息编码** - 颜色表示销售占比,气泡大小表示订单数量
3. **交互式详情** - 鼠标悬停显示具体数值和百分比
4. **响应式设计** - 地图支持缩放和拖动
5. **动画效果** - Top 5城市有波纹动画效果
