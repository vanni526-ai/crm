## 导入方式

由于ICS文件较大(8.9MB,Base64编码后12MB),通过前端上传会导致HTTP请求超时或被截断。因此采用**服务器端直接导入**的方式:

```bash
npx tsx import-ics-server.mjs
```

## 数据映射

从ICS事件到schedules表的字段映射:

| ICS字段 | schedules字段 | 处理逻辑 |
|---------|--------------|----------|
| summary | deliveryCourse, courseType | 完整课程描述 |
| summary (第一个空格前) | customerName | 提取客户名 |
| organizer.params.CN | deliveryTeacher, teacherName | 提取老师名称 |
| start | startTime, classDate, classTime | 提取日期和时间 |
| end | endTime, classTime | 提取结束时间 |
| location | location | 上课地点 |
| description | notes | 课程备注 |
