# 月度日历看板功能说明

## 功能概述

在数据导入页面的课程日程看板中新增了**月度日历**视图,提供传统的月历网格布局,方便用户直观地查看每月的课程安排。

## 功能特性

### 1. 三种视图模式
- **列表视图** - 表格形式展示所有课程
- **日历视图** - 按日期分组展示课程详情
- **月度日历** - 传统月历网格布局(新增)

### 2. 月度日历布局
- **星期标题** - 日、一、二、三、四、五、六
- **日历网格** - 7列x5-6行的标准月历布局
- **空白单元格** - 月初和月末的空白日期显示为灰色背景

### 3. 课程显示
- **课程卡片** - 每个日期单元格中显示当天的课程
- **显示格式** - `时间 客户名`(如"12:00 虞霓")
- **最多显示3个** - 每个单元格最多显示3个课程
- **更多提示** - 超过3个课程时显示"+N 更多"

### 4. 月份导航
- **上一月按钮** - 左侧箭头按钮,切换到上一个月
- **下一月按钮** - 右侧箭头按钮,切换到下一个月
- **月份标题** - 中间显示"YYYY年 MM月"

### 5. 今日高亮
- **背景色** - 今天的日期单元格使用浅蓝色背景
- **边框** - 今天的日期使用主题色边框
- **日期数字** - 今天的日期数字使用主题色文字

## 使用场景

1. **月度课程规划** - 快速查看整月的课程安排密度
2. **课程冲突检测** - 直观发现同一天课程过多的情况
3. **老师排班管理** - 查看老师在整月的工作分布
4. **客户预约查询** - 快速找到特定日期的课程安排

## 技术实现

### 日历网格生成算法
```typescript
const year = currentMonth.getFullYear();
const month = currentMonth.getMonth();
const firstDay = new Date(year, month, 1).getDay(); // 月初是星期几
const daysInMonth = new Date(year, month + 1, 0).getDate(); // 当月天数

// 填充空白单元格(月初)
for (let i = 0; i < firstDay; i++) {
  cells.push(<空白单元格 />);
}

// 填充日期单元格
for (let day = 1; day <= daysInMonth; day++) {
  const daySchedules = 筛选当天的课程;
  cells.push(<日期单元格 day={day} schedules={daySchedules} />);
}
```

### 课程数据匹配
```typescript
const daySchedules = schedules.filter((schedule) => {
  if (!schedule.classDate) return false;
  const scheduleDate = new Date(schedule.classDate);
  return scheduleDate.toLocaleDateString() === currentDate.toLocaleDateString();
});
```
